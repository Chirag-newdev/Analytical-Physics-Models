# Water Molecule — Orbital Visualization & Bond Length Analysis

Quantum chemistry calculations on a water molecule (H₂O) using **Psi4** and **STO-3G / 6-31G\*** basis sets. Computes molecular orbitals, visualizes HOMO/LUMO using VMD, and studies how O-H bond length affects total energy under HF and MP2 methods.

---

## Background

Water has 10 electrons occupying 5 molecular orbitals (MOs) in its ground state. These orbitals are obtained by solving the **Restricted Hartree-Fock (RHF)** equations:

$$
\hat{F} \psi_i = \epsilon_i \psi_i
$$

where $\hat{F}$ is the Fock operator, $\psi_i$ are the molecular orbitals, and $\epsilon_i$ are the orbital energies (eigenvalues).

The geometry used throughout:

| Atom | X (Å) | Y (Å) | Z (Å) |
|------|--------|--------|--------|
| O    | 0.000  | 0.000  | −0.066 |
| H    | 0.000  | −0.759 | 0.522  |
| H    | 0.000  | 0.759  | 0.522  |

Bond angle: **104.5°**, O-H bond length: **0.96 Å**

## Files

| File | Description |
|------|-------------|
| `HOMO_LUMO.py` | Runs SCF with STO-3G, prints HOMO/LUMO energies, generates `.cube` files for VMD visualization |
| `water_geometry.py` | SCF calculation with cc-pVDZ basis, prints dipole moment |
| `water_bond_vary_effects.py` | Scans O-H bond length from 0.7–2.0 Å, computes HF and MP2 energies, plots the difference |
| `output.dat` | Full Psi4 SCF log from the STO-3G calculation |
| `geom.xyz` | Optimized water geometry |
| `Psi_a_*.cube` | Alpha MO cube files (1–7), generated by Psi4 for VMD |
| `HOMO.cube` / `LUMO.cube` | Cube files specifically for HOMO (MO 5) and LUMO (MO 6) |

## Running the Scripts

**Requirements:** [Psi4](https://psicode.org/), NumPy, Matplotlib

```bash
# Generate HOMO/LUMO cube files and print orbital energies
python HOMO_LUMO.py

# Scan bond length and compare HF vs MP2
python water_bond_vary_effects.py

# Basic SCF with dipole moment
python water_geometry.py
```

---

## Visualizing Orbitals in VMD

Once cube files are generated:

1. Open VMD → `File → New Molecule` → load `geom.xyz`
2. `File → Load Data Into Molecule` → load `HOMO.cube`
3. In **Graphics → Representations**:
   - Drawing Method: `Isosurface`
   - Isovalue: `0.05` (adjust for desired orbital lobe size)
   - Draw: `Solid Surface`
4. Repeat for `LUMO.cube` to compare both orbitals

> **Tip:** Use two separate representations with different colors (e.g. blue/red) for positive and negative lobes of the wavefunction.

## Bond Length Scan (HF vs MP2)

`water_bond_vary_effects.py` varies the O-H bond length from **0.7 Å to 2.0 Å** in 80 steps, computing both SCF (HF) and MP2 energies at each step using a **6-31G\*** basis set, then plots their difference.

The energy minimum corresponds to the equilibrium bond length. MP2 adds electron correlation corrections on top of HF:

$$
E_{\text{MP2}} = E_{\text{HF}} + E_{\text{corr}}^{(2)}
$$

## Notes

- All calculations use the **STO-3G** minimal basis set unless otherwise specified  results are qualitatively correct but not quantitatively accurate
- The `6-31G*` basis in `water_bond_vary_effects.py` is more reliable for energy comparisons
- Cube files are large  not committed to the repo. Regenerate them by running `HOMO_LUMO.py`
